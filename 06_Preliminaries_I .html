6.1.1 Groups, Abelian Groups 群、アーベル群

We first define a group without going into group theory
まず群論に立ち入ることなく群を定義する

Definition 6.1 (Group) . A group is a set \(G\) with a binary operation, ∗, such that the operation is associative :
定義 6.1 (群)　群とは、二項演算 ∗ を持つ集合 \(G\) であり、その演算は結合的である。 \[ (a ∗ b) ∗ c = a ∗ (b ∗ c) \] for all \(a, b, c ∈ G\), it has an identity element \(e\):
すべての \(a, b, c ∈ G\) に対して、単位元 \(e\) を持つ。 \[ a ∗ e = a = e ∗ a \] for all \(a ∈ G\), and finally, every \(a ∈ G\) has an inverse element \(b\), such that
すべての\(a ∈ G\)に対して、逆元\(b\)があり、 \[ a ∗ b = e = b ∗ a. \]

The main property of emphasis in a group is that everything is invertible. For example this means that if we can add, we can also subtract, and if we can multiply, we can also divide. Associativity is in some sense present in everything, while even if something doesn’t have an identity, one can usually just add one artificially.
群における強調される主な性質は、すべての要素が可逆であるというこです。例えば、足し算ができれば引き算もでき、掛け算ができれば割り算もできるということです。結合法則は、ある意味ではあらゆるものに存在し、たとえ何かが単位元を持たないとしても、通常は人為的に単位元を加えるだけで済みます。

Definition 6.2 (Subgroup) . A subgroup \(H\) of a group \(G\) is a subset that is closed under the operation of that whole group:
定義 6.2 (部分群) 群 \(G\) の部分群 \(H\) とは、その群全体の作用素に関して閉じた部分集合のことである。 \[ h_1h_2 ∈ H \] for all \(h_1, h_2 ∈ H\) and is itself a group under this restricted operation.
すべての\(h_1, h_2∈H\)に対して成り立ち、それ自体はこの制限された操作の下で群である。

Example 6.1. Given a set \(X\), the set of invertible functions \(f : X → X\) with operation as composition of functions is a key example of a group.
例 6.1. 集合 \(X\) が与えられたとき、関数の合成として演算される可逆関数 \(f : X → X\) の集合は、群の重要な例です。

We see that in this example there is exactly one identity function and every function has a unique inverse. This is true in general:
この例では、恒等関数は1つだけ存在し、すべての関数には一意の逆関数が存在することがわかります。これは一般に当てはまります。

Lemma 6.1. The identity and inverses are uniquely defined.
補題 6.1. 恒等関数と逆関数は一意に定義されます。

Proof. Identity is Unique:
証明。恒等関数は一意である:

Suppose \(e\) and \(e^\prime\) are identities.
\(e\) と \(e^\prime\) が恒等式であるとします。 \[ e = e ∗ e^\prime = e^\prime \] Inverse is Unique:
Suppose \(b\) has inverse \(c\), and another inverse \(d\).
逆関数は一意である:
\(b\) に逆関数 \(c\) があり、さらに別の逆関数 \(d\) があるとします。 \[ c = c ∗ e = c ∗ (b ∗ d) = ( c ∗ b) ∗ d = e ∗ d = d. \]

If the operation can be carried out in either order then we have an Abelian group:
どちらの順序でも演算を実行できる場合は、アーベル群が得られます。

Definition 6.3 (Abelian Group) . An Abelian group is a group \(G\), ∗ whose operation ∗ is commutative :
定義 6.3 (アーベル群) アーベル群とは、その演算 ∗ が可換である群 \(G\), ∗ のことである。 \[ a ∗ b = b ∗ a \] for all \(a, b ∈ G\). It is common for Abelian groups to have ∗ written as + and the operation called addition with identity 0.
すべての \(a, b ∈ G\) に対して成り立つ。アーベル群では、∗ は + と書かれ、演算は恒等式 0 の加法と呼ばれるのが一般的である。

Example 6.2. The set of integers \(\mathbb{Z}\) together with addition is a key example of an Abelian group.
例 6.2. 整数の集合 \(\mathbb{Z}\) と加法は、アーベル群の重要な例です。

Example 6.3. The set integers modulo \(n\), \(\mathbb{Z}/n\mathbb{Z}\), together with addition is another key example of an Abelian group. It is an example of a quotient group of \(\mathbb{Z}\) by \(n\mathbb{Z}\), which can be thought of as a group formed by setting the elements of the subgroup \(n\mathbb{Z}\) to zero in \(\mathbb{Z}\) along with its consequences (for example, if two numbers differ by \(n\), then they are now equal). A similar construction can be made for any abelian group and one of its subgroups.
例 6.3. 整数の集合を法 \(n\) で割ったもの \((\mathbb{Z}/n\mathbb{Z}\)）と加法の組み合わせは、アーベル群のもう一つの重要な例です。これは、\(\mathbb{Z}\) を部分群 \(n\mathbb{Z}\) で割った剰余群の例であり、\(n\mathbb{Z}\) の要素を \(\mathbb{Z}\) の中で \(0\) とみなすことで構成される群と考えることができます（例えば、2つの数が \(n\) だけ異なる場合、それらは同じと見なされます）。同様の構成は、任意のアーベル群とその部分群に対しても可能です。

Example 6.4. The set of symmetries of an equilateral triangle (ways of rotating and reflecting it which preserve its appearance) form a group under the operation of composition, i.e. doing one symmetry after another, but it is not an abelian group. For instance, horizonally reflecting the triangle and then rotating it \(120^\circ\) clockwise is not the same symmetry as rotating it \(120^\circ\) clockwise and then horizontally reflecting it, so the operation is not commutative.
例 6.4. 正三角形の対称性の集合（見た目を保ったまま回転や鏡映を行う方法）は、合成操作、つまり対称性を次々と行うことによって群を形成しますが、アーベル群ではありません。例えば、三角形を水平に鏡映してから時計回りに \(120^\circ\) 回転させる操作と、時計回りに \(120^\circ\) 回転してから水平に鏡映する操作は同じ対称性ではありません。したがって、この操作は可換ではありません。

Definition 6.4 (Cyclic Group) . A finite cyclic group is a group with a finite number of elements which can all be reached by repeatedly applying the group operation on one element, known as the generator . For example, the set of integers modulo \(n: \{0,1,2, ..., n − 1\}\) denoted \(\mathbb{Z}/n\mathbb{Z}\) is a finite cyclic group under addition with generator \(1\), while \(\mathbb{Z}\),+ is an example of an infinite cyclic group.
定義 6.4 (巡回群) 有限巡回群とは、生成元と呼ばれる1つの元に群演算を繰り返し適用することで、すべての元に到達できる有限個の元を持つ群である。例えば、\(n: \{0,1,2, ..., n − 1\}\) を法とする整数の集合 \(\mathbb{Z}/n\mathbb{Z}\) は、生成元 \(1\) を持つ加法による有限巡回群である。一方、\(\mathbb{Z}\),+ は無限巡回群の例である。

  • • One of our key checkpoints is to show that the sum of points defined in the outline makes an elliptic curve into an Abelian group.
    私たちの重要なチェックポイントの 1 つは、アウトラインで定義された点の合計によって楕円曲線がアーベル群になることを示すことです。

  • • Notice \(\mathbb{Z}/n\mathbb{Z}\) has the property that adding \(1\) \(n\) times gets back to \(0\), this is an example of torsion , which can be defined for any Abelian group:
    • \(\mathbb{Z}/n\mathbb{Z}\) には \(1\) を \(n\) 回加算すると \(0\) に戻るという性質があります。これは任意のアーベル群に対して定義できるねじれの例です。

Definition 6.5 (Scalar Multiplication by Integers) . Let \(G\) be an Abelian group. For \(n ∈ Z\) and \(g ∈ G\), define \(ng\) by
定義 6.5 (整数によるスカラー乗算) \(G\) をアーベル群とする。\(n ∈ Z\) および \(g ∈ G\) に対して、\(ng\) を次のように定義する。 \[ ng = \underbrace{g + g + · · · + g}_{n回} \] when \(n ≥ 0\) and \((−n)(−g)\) when \(n <0\).
\(n ≥ 0\) の場合。\(n <0\)の場合は, \((−n)(−g)\)。

Now we have the following definition:
ここで次の定義が得られます。

Definition 6.6 (Torsion Subgroup). Given an Abelian group \(G\), + and an integer \(n\), the set
定義 6.6 (捩れ部分群) アーベル群 \(G\)、+、整数 \(n\) が与えられたとき、集合 \[ G[n] = \{g ∈ G : ng = 0 \} \] is a group under the same addition, called the \(n\)-torsion subgroup. The torsion subgroup of \(G\) is
は同じ加法による群であり、\(n\)-捩れ部分群と呼ばれる。\(G\)の捩れ部分群は \[ G = \bigcup_n G[n] \] which consists of all the torsion elements of the group.
これは、群のすべてのねじれ要素で構成されます。

Our key milestone for encountering a torsion subgroup is when we get the torsion points of an elliptic curve.
ねじれ部分群に遭遇する重要なマイルストーンは、楕円曲線のねじれ点を取得するときです。

Here is a simple construction to build bigger Abelian groups:
より大きなアーベル群を構築するための簡単な構成は次のとおりです。

Definition 6.7. Give two Abelian groups \(A\) and \(B\), define an Abelian group \(A⊕B\) by taking the set \(A×B\) of ordered pairs \((a,b)\) with \(a ∈ A\) and \(b ∈ B\) and define addition pointwise :
定義 6.7. 2 つのアーベル群 \(A\) と \(B\) を与え、\(a ∈ A\) および \(b ∈ B\) である順序付きペア \((a,b)\) の集合 \(A×B\) をとってアーベル群 \(A⊕B\) を定義し、点ごとに加法を定義します。 \[ (a_1, b_1) + (a_2, b_2) = (a_1 + a_2, b_1+ b_2) \] for all \(a_1, a_2 ∈ A\) and \(b_1, b_2 ∈ B\).
すべての \(a_1, a_2 ∈ A\) と \(b_1, b_2 ∈ B\) に対して。

We will use this to show many key groups associated with elliptic curves are just two copies of the same group, for example \(Z/n ⊕ Z/n\).
これを用いて、楕円曲線に関連する多くの重要な群が、同じ群の2つのコピーであることを示します。(例: \(Z/n ⊕ Z/n\))。

6.1.2 Rings and Fields 環と体

We can also get two operations on the same set:
同じ集合に対して 2 つの操作を実行することもできます。

Definition 6.8 (Ring) . A ring is a set R with two operations, usually called addition + and multiplication ×, satisfying several properties:
定義 6.8 (環) 環とは、通常加算 + と乗算 × と呼ばれる2つの演算を持つ集合 R であり、いくつかの性質を満たす。

　\(R\), + is an Abelian group. 　\(R\), × is associative, and together distribute :
　\(R\), + はアーベル群である。 　\(R\), × は結合的であり、一緒に分配する。 \[ \begin{align} &a × (b + c) = a × b + a × c, \\ \\ &(a + b) × c = a × c + b × c \end{align} \] for all \(a, b, c ∈ R\). If the multiplication is commutative then the ring is called a commutative ring . Often rings will also have an identity element for multiplica- tion included as part of the definition.
すべての \(a, b, c ∈ R\) に対して成り立つ。乗法が可換な場合、その環は可換環と呼ばれる。多くの場合、環には定義の一部として乗法の単位元も含まれる。

In other words, rings have a notion of addition, subtraction and multiplication but not necessarily division. When there is a notion of division, they are called division rings and when the multiplication is commutative, we get a field :
言い換えれば、環は加法、減法、乗法の概念を持ちますが、必ずしも除法の概念を持つわけではありません。除法の概念を持つ環は除算環と呼ばれ、乗法が可換な場合は体が得られます。

Definition 6.9 (Field) . A field F is a commutative ring with the added property that every non-zero element has a multiplicative inverse.
定義 6.9 (体) 体 F は、すべての非ゼロ元に逆元が存在するという追加の特性を持つ可換環である。

In other words, \(F\), + is an Abelian group and \(F \{0\}\),× is an Abelian group, with distributivity holding them together.
言い換えれば、\(F\), + はアーベル群であり、\(F \{0\}\),× は分配法則によって結び付けられたアーベル群です。

Definition 6.10 (Characteristic of a Field) . The characteristic of a field is the smallest \(n\) for which scaling \(1\) by \(n\) equals zero, that is \(n(1) = 0\). The characteristic is defined to be zero if no \(n\) exists.
定義 6.10 (体の標数) 体の標数とは、\(1\) を \(n\) でスケーリングしてゼロとなる最小の \(n\)、つまり \(n(1) = 0\) のことである。\(n\) が存在しないとき、標数はゼロと定義される。

Proposition 6.1. If the characteristic is non-zero, then it is prime.
命題 6.1. 特性がゼロでない場合、それは素数です。

Proof. Suppose for the sake of contradiction that we have a field F of character- istic \(ab\) where \(a, b ∈ \mathbb{N}_{≥2}\). We know that every element in \(F\) has a multiplicative inverse. Suppose \(a^{-1}\) is the inverse of \(a\). Therefore, \(1 = aa^{−1}\) from which multiplying by \(b\) implies that \(b = baa^{−1}= 0\). This is impossible because \(0 = b < ab\) which is assumed to be the smallest value that can scale \(1\) to become \(0\). This contradiction necessitates that the characteristic cannot be composite if it is nonzero.
証明。矛盾を避けるために、特性 \(ab\) の体 F があり、\(a, b ∈ \mathbb{N}_{≥2}\) であるとする。\(F\) のすべての元には逆元があることはわかっている。\(a^{-1}\) が \(a\) の逆元であるとする。したがって、\(1 = aa^{−1}\) となり、これに \(b\) を掛けると \(b = baa^{−1}= 0\) となる。これは不可能である。なぜなら、\(0 = b < ab\) は \(1\) を \(0\) にスケールできる最小の値であると仮定されるからである。この矛盾により、特性が 0 でない場合、合成数は存在しないことになる。

Example 6.5. \(\mathbb{Z}\) with addition and multiplication is a commutative ring and so is \(\mathbb{Z}/n\mathbb{Z}\) with addition and multiplication modulo \(n\).
例 6.5. 加算と乗算を伴う \(\mathbb{Z}\) は可換環であり、\(n\) を法とする加算と乗算を伴う \(\mathbb{Z}/n\mathbb{Z}\) も可換環です。

Example 6.6. \(\mathbb{Q}\) the set of rational numbers together with addition and mul- tiplication is a field of characteristic zero.
例 6.6. \(\mathbb{Q}\) 有理数の集合は加法と乗法を合わせた特性が 0 の体である。

Example 6.7. \(\mathbb{C}\) the set of complex numbers together with addition and multiplication is a field of characteristic zero.
例 6.7. \(\mathbb{C}\) 複素数とその加算および乗算を合わせた集合は、特性 0 の体である。

Example 6.8. \(\mathbb{Z}/p\mathbb{Z}\) the set of integers modulo a prime \(p\) together with addition and multiplication modulo p is a field. In this context, we will rewrite this as \8\mathbb{F}_p\) to emphasise it is a finite field of size p. It is a field of characteristic \(p\).
例 6.8. 素数 \(p\) を法とする整数の集合 \(\mathbb{Z}/p\mathbb{Z}\) は、p を法とする加法および乗法を合わせた体である。この文脈では、これがサイズ p の有限体であることを強調するため、これを \8\mathbb{F}_p\) と書き直す。これは標数 \(p\) の体である。

このサブセクションでは、スカラー乗算と加算の相互作用について説明します。

すべての\(λ∈R, u∈U, v∈V\)に対して。

例6.11. \(\mathbb{F}_p ⊕ \mathbb{F}_p\)は\(\mathbb{F}_p\)ベクトル空間である。

例 6.12. \(\mathbb{Z}/n\mathbb{Z} ⊕ \mathbb{Z}/n\mathbb{Z}\) は \(\mathbb{Z}/n\mathbb{Z}\) 加群である

6.2 Morphisms in Abstract Algebra 抽象代数における射影

So far, we’ve introduced several key objects in abstract algebra: groups , rings , fields , and modules . Each of these structures is defined by a set equipped with operations satisfying certain axioms. However, we’ve yet to discuss the concept of morphisms — the functions between these structures that preserve their defining properties.
これまで、抽象代数学におけるいくつかの重要な対象、すなわち群、環、体、加群を紹介してきました。これらの構造はそれぞれ、特定の公理を満たす演算を備えた集合によって定義されます。しかし、射の概念、つまりこれらの構造間の定義特性を保持する関数についてはまだ説明していません。

Understanding morphisms is important because they allow us to compare al- gebraic objects meaningfully. They also provide powerful shortcuts: instead of verifying all properties directly, we can often reason about structures via the morphisms between them.
射を理解することは、代数的対象を意味のある形で比較することを可能にするため重要です。また、射は強力な近道も提供します。つまり、すべての性質を直接検証する代わりに、それらの間の射を介して構造について推論できる場合が多いのです。

Before diving into the details, getting the big idea can save a lot of effort, because to write everything in every detail can be difficult to get right com- pletely, and it can be overwhelming sometimes when you’re just starting out.
詳細に入る前に、全体的なアイデアを把握することで、多くの労力を節約できます。すべてを細部まで書き出すのは難しく、始めたばかりのときは圧倒されてしまうことがあるからです。

Morphisms that we discuss are functions between objects of the same type that preserve their defining properties. Even if it appears they are defined to not preserve some properties, they usually are able to be shown as a consequence of the preserved ones that are included.
ここで論じる射とは、同じ型のオブジェクト間の、定義特性を保持する関数です。たとえ一部の特性を保持しないように定義されているように見えても、通常は、含まれる保持特性の結果として、それらの特性を示すことができます。

6.2.1 Group Homomorphisms 群準同型

Definition 6.13 (Group Homomorphisms) . Given two groups \(G\) and \(H\), a group homomorphism is a function
定義 6.13 (群準同型) 2つの群 \(G\) と \(H\) が与えられたとき、群準同型とは関数である。 \[ \varphi : G → H \] that satisfies
以下を満足する \[ \varphi(ab ) = \varphi(a)\varphi(b)　for　all　a, b ∈ G \] Definition 6.14 (Group Isomorphism) . A group isomorphism is a bijective homomorphism whose inverse is also a homomorphism. If such a map exists we say that \(G\) and \(H\) are isomorphic as groups — essentially, they are the same group structure up to renaming of elements. \(G\) and \(H\) being isomorphic is denoted by \(G \simeq H\).
定義 6.14 (群同型) 群同型とは、その逆写像も準同型となるような全単射準同型である。そのような写像が存在する場合、\(G\) と \(H\) は群として同型であると言う。つまり、本質的には、元の名前を変えない限り、それらは同じ群構造である。\(G\) と \(H\) が同型であることは、\(G \simeq H\) と表記される。

Definition 6.15 (Automorphism) . An automorphism is an isomorphism of a group to itself.
定義 6.15 (自己同型) 自己同型とは、群とそれ自身との同型性である。

You may wonder if \(\varphi\) should send the identity of \(G\) to the identity of \(H\) but this is a consequence of preserving the operation.
\(\varphi\) は \(G\) の恒等式を \(H\) の恒等式に送るべきかどうか疑問に思うかもしれませんが、これは操作を保存した結果です。

Example 6.13. Group homomorphisms \(f\) between Abelian groups might be written
例6.13. アーベル群間の群準同型\(f\)は次のように書ける。 \[ f(a + b) = f(a) + f(b), \] which is just an additive function. We know that \(f(0) = 0\) and \(f(−a) = −a\) are consequences of additivity.
これは単なる加法関数です。\(f(0) = 0\) と \(f(−a) = −a\) は加法性の結果であることが分かっています。

6.2.2 Homomorphisms of Rings and Modules 環と加群の準同型

  • ● For rings , a ring homomorphism and multiplication: \(\varphi : R → S\) must preserve both addition
    環 に対して、環準同型と乗法: \(\varphi : R → S\) は、加法と乗法の両方を保存する必要がある。 \[ \varphi(a + b) = \varphi(a) + \varphi(b),　　\varphi(ab) = \varphi(a)\varphi(b) \] and often (but not always), \(\varphi(1_R) = 1_S\).
    そして多くの場合（常にではないが）、\(\varphi(1_R) = 1_S\)となる。

    Example 6.14. An illustrative example is the ring homomorphism \(φ : \mathbb{Z} → \mathbb{Z}/n\mathbb{Z}\). This holds as addition and multiplication are preserved in modular arithmetic.
    例 6.14. わかりやすい例として、環準同型 \(φ : \mathbb{Z} → \mathbb{Z}/n\mathbb{Z}\) が挙げられます。これは、モジュラー算術において加算と乗算が保存されるので成立します。
    Example 6.15. The most familiar example of a field automorphism on \(\mathbb{C}\) might be that of complex conjugation:
    例 6.15. \(\mathbb{C}\) 上の体自己同型の最もよく知られた例は複素共役の例でしょう。 \[ z = x + yi \mapsto \overline{z} = x − yi \]

    Definition 6.16. Given a field \(F\), the set of all field automorphisms is denoted \(Aut(F)\) and is itself a group under function composition.
    定義 6.16. 体 \(F\) が与えられたとき、すべての体自己同型全体の集合は \(Aut(F)\) と表され、それ自体が関数合成による群である。


  • ● For modules , a module homomorphism between two \(R\)-modules \(M\) and \(N\) is a function \(f : M → N\) that preserves addition and scalar multiplication:
    加群について、2つの \(R\)-加群 \(M\) と \(N\) 間の加群準同型は、加算とスカラー乗算を保存する関数 \(f : M → N\) です。 \[ f(x + y) =f(x) +f(y),　f(rx) = rf(x)　for　all　x, y ∈ M, r ∈ R \]

    Definition 6.17. This is also an example but is used later too: an \(F\)-vector space \(V\) is two dimensional if
    定義 6.17 これは例ですが、後ほども使われます。\(F\)-ベクトル空間 \(V\) が2次元であるとは、 \[ V \simeq F ⊕ F \]
