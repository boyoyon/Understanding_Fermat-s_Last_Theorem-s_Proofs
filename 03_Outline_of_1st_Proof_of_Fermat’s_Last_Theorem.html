Understanding Fermat’s Last Theorem’s Proofs
フェルマーの最終定理の証明を理解する

3 Outline of 1st Proof of Fermat’s Last Theorem フェルマーの最終定理の第一証明の概要

3.1 Initial Summary 初期概要

We do not assume the reader is familiar with the following key terms, which will be introduced here: Frey’s curve, Elliptic curve, Modular.
ここで紹介する「フライ曲線」、「楕円曲線」、「モジュラー」といったキーワードについては、読者がすでにご存知ないことを前提としています。

We will first give an outline of the structure of the first proof of FLT without elaborating on the term Modular nor mentioning Galois Representations .
まず、「モジュラー」という用語の詳細やガロア表現への言及をせずに、FLT の最初の証明の構造の概要を説明します。

  • • Cases \(n = 3 ,4\) are done using elementary methods
    ケース \(n = 3,4\) は基本的な方法を使用して実行されます

  • • Assume FLT is false and define Frey’s curve.
    FLT が偽であると仮定し、フライ曲線を定義します。

  • • Ribet proved that Frey’s curve is not modular.
    リベットはフライの曲線がモジュラーではないことを証明した。

  • • Wiles proved that Frey’s curve is modular.
    ワイルズはフライの曲線がモジュラーであることを証明した。

  • • Contradiction, hence Fermat’s Last Theorem is true.
    矛盾。したがってフェルマーの最終定理は真です。

3.1.1 Cases \(n = 3 ,4\) \(n = 3,4\) の場合

The proofs of these are in the Appendix (12.1.3,12.1).
これらの証明は付録（12.1.3、12.1）にあります。

We explain why it now suffices to consider the case when \(n\) is an odd prime \(P ≥ 5\):
\(n\) が奇数の素数 \(P ≥ 5\) の場合を考えれば十分である理由を説明します。

Theorem 3.1. If Fermat’s Last Theorem is proved true when \(n\) is an odd prime \(P ≥ 5\) or \(4\) and \(a, b, c\) are pairwise coprime, it must also be true in general. Proof. We show a solution for \(n > 2\) gives a solution for \(n = 4\) or \(n\) and odd prime.
定理3.1. フェルマーの最終定理は、\(n\) が奇数の素数 \(P ≥ 5\)、または \(4\) と \(a, b, c\) が互いに素な場合に成立することが証明されれば、一般にも成立する。証明。\(n > 2\) の解は、\(n = 4\)、または \(n\) と奇数の素数の場合の解を与えることを示す。

We will do two cases: \(n\) is a power of 2 and \(n\) is not a power of 2:
ここでは、\(n\) が 2 の累乗である場合と、\(n\) が 2 の累乗でない場合の 2 つのケースを取り上げます。

  • 1. Suppose that \(n = 2^k\) satisfies Fermat’s equation for some \(k ∈ \mathbb{Z}_{≥2}\). We have
    \(n = 2^k\) が何らかの \(k ∈ \mathbb{Z}_{≥2}\) に対してフェルマー方程式を満たすと仮定する。 \[ a^{2^k}+ b^{2^k}= c^{2^k} \] \[ \Longleftrightarrow (a^{2^{k−2}})^4 +(b^{2^{k−2}})^4 = (c^{2^{k−2}})^4 \] But the case \(n = 4\) has no non-zero solutions.
    しかし、\(n = 4\) の場合、非ゼロの解は存在しません。

  • 2. Now suppose that \(n = pr\) satisfies Fermat’s equation for some \(r ∈ Z^+\) and odd prime \(p\). Then
    ここで、\(n = pr\) が、ある \(r ∈ Z^+\) と奇素数 \(p\) に対してフェルマー方程式を満たすと仮定する。すると、 \[ a^{pr} + b^{pr} = c^{pr} \] \[ \Longleftrightarrow (a^r)^p + (b^r)^p = (c^r)^p \] which is only true if \(n = p\) satisfies Fermat’s equation.
    これは\(n = p\)がフェルマー方程式を満たす場合にのみ真です。

Since we know the \(n = 3\) case, we can assume \(P ≥ 5\).
\(n = 3\) の場合がわかっているので、\(P ≥ 5\) と仮定できます。

Now if any pair of \(a, b, c\) satisfying the equation share a prime factor, then so does the third and we can just divide the n-th power of this prime from the equation. We can do this until no such common factors exist.
ここで、方程式を満たす \(a, b, c\) の任意のペアが共通の素因数を持つ場合、3番目の素因数も同様に持つので、この素因数のn乗を方程式から割るだけで済みます。このような共通因数がなくなるまで、これを繰り返します。

3.1.2 Frey’s Curve and its Properties フライ曲線とその特性

Keywords: Discriminant, Non-Singular, Good/Bad/Additive/Multiplicative Re- duction, Semistable.
キーワード: 判別式、非特異値、良/悪/加法/乗法縮減、半安定。

Assume FLT is false:
FLT が偽であると仮定します。

That is, there exists \(a, b, c\) pairwise coprime, and a prime \(P ≥ 5\) prime, such that
つまり、互いに素な\(a, b, c\)と素な\(P ≥ 5\)が存在し、 \[ a^P + b^P = c^P \]

Frey’s curve [7] is then defined from the equation
フライの曲線[7]は次の式から定義される。 \[ y^2 = x(x − a^P)(x + b^P) \] and is an example of an Elliptic Curve .
これは楕円曲線の例です。

We can temporarily take as a definition of an elliptic curve those equation of the form \(y^2\) equals a cubic in \(x\) with integer coefficients.
楕円曲線の定義として、\(y^2\) が整数係数を持つ \(x\) の 3 次方程式に等しいという形式をとることができます。

The reader may be familiar with the concept of a discriminant, especially in regard to quadratic equations. If we have a quadratic expression:
読者の皆様は判別式の概念、特に二次方程式に関する概念に馴染みがあるかもしれません。次の二次式があるとします。 \[ (x − r_1)(x − r_2) = x^2 − (r_1 + r_2)x + r_1r_2 \] then the discriminant is
判別式は \[ (r_1 + r_2)^2 − 4r_1r_2 = r_1^2 − 2r_1r_2 + r_2^2 = (r_1 − r_2)^2 \] which is the square of the difference of the roots. This is defined for cubics by:
これは根の差の2乗です。これは3次方程式では次のように定義されます。

Definition 3.1 (Discriminant). Suppose a cubic polynomial has roots \(r_1, r_2, r_3,\) with leading coefficient 1, then its discriminant is given by
定義 3.1 (判別式) 3次多項式が根 \(r_1, r_2, r_3,\) を持ち、その係数が1であるとすると、その判別式は次のように与えられる。 \[ ∆ = (r_1 − r_2)^2(r_1 − r_3)^2(r_2 − r_3)^2 \]

We also take this to be the discriminant of the elliptic curve
これを楕円曲線の判別式とみなす。 \[ y^2 = (x − r_1)(x − r_2)(x − r_3) \] \(∆\) detects whether or not the roots coincide. Part of the definition of an el- liptic curve is that it is non-singular , which means that none of these roots coincide or equivalently
\(∆\) は、根が一致するかどうかを判定します。楕円曲線の定義の一部は、非特異性を持つことであり、これはこれらの根がどれも一致しない、あるいは同値ではないことを意味します。 \[ ∆ \neq 0 \]

Proposition 3.1.　Frey’s curve is non-singular. Moreover,
命題3.1. フライ曲線は特異ではない。さらに、 \[ ∆ = ( abc )^{2p} \] Proof. For Frey’s Curve, \(r_1 = 0 , r_2= a^P, r_3= −b^P\). Hence
証明。フライ曲線では、\(r_1 = 0 , r_2= a^P, r_3= −b^P\)である。したがって、 \[ ∆ = (a^P − 0)^2(−b^P− 0)^2(−b^P− a^P)^2 = ( abc )^{2P} \neq 0 \]

In analysing such equations it can be helpful to reduce it modulo a prime number \(p\), i.e. setting \(p = 0\) which turns this into a finite problem that can be understood explicitly. Given a prime \(p\), one can consider the reduction at \(p\) of the Elliptic Curve by reducing all the coefficients of the curve modulo \(p\). One issue that arises is that the \(∆\) might be divisible by p and the new equation becomes singular.
このような方程式を解析する際には、素数 \(p\) を法として簡約すると便利です。つまり、\(p = 0\)と設定することで、この問題を明示的に理解できる有限問題に変換できます。素数\(p\)が与えられた場合、楕円曲線のすべての係数を\(p\)を法として簡約することで、\(p\)における楕円曲線の簡約を検討できます。ここで生じる問題の一つは、\(∆\)がpで割り切れる可能性があり、新しい方程式が特異になる可能性があることです。

Definition 3.2.　If \(p\) does not divide \(∆\), the discriminant is non-zero modulo \(p\), which we call good reduction at \(p\). If \(p | ∆\), the discriminant is 0 modulo \(p\), which we call bad reduction at \(p\). There are two types of bad reduction: multiplicative reduction is if exactly two roots coincide and additive reduction is if all three roots coincide.
定義 3.2. \(p\) が \(∆\) を割り切れない場合、判別式は \(p\) を法として非ゼロとなり、これを \(p\) における良好な縮約と呼びます。\(p | ∆\) の場合、判別式は \(p\) を法として 0 となり、これを \(p\) における不良縮約と呼びます。不良縮約には2種類あります。乗法縮約はちょうど2つの根が一致する場合、加法縮約は3つの根がすべて一致する場合です。

(蛇足)
\(p | ∆\) ･･･ \(p\) は \(Δ(判別式)\) で割り切れる。
\(p ∤ ∆\) ･･･ \(p\) は \(Δ(判別式)\) で割り切れない
(蛇足おわり)

Definition 3.3 (Semi-stable).　We call an elliptic curve semi-stable if it has only good or multiplicative reduction at every prime, or equivalently it never has additive reduction at a prime.
定義 3.3 (半安定)　楕円曲線が、すべての素数において良好な縮約または乗法的な縮約のみを持つ場合、または同値として、素数において加法的な縮約がまったく持たない場合、その楕円曲線は半安定であると呼ばれます。

Proposition 3.2. Frey’s curve is semistable.
命題3.2. フライ曲線は半安定である。

Proof. Since \(a, b, c\) are coprime, if a prime divides \(a^P\), it does not divide \(b^P\). This means when reducing it any prime the roots \(a^P\) and \(b^P\) never coincide, hence the reduction at any prime is not additive.
証明。\(a, b, c\)は互いに素なので、素数が\(a^P\)を割り切る場合、\(b^P\)は割り切れません。つまり、\(a^P\)を任意の素数に約分しても、根\(a^P\)と\(b^P\)は決して一致しません。したがって、任意の素数における約分は加法性を持ちません。

3.1.3 Ribet and Wiles リベットとワイルズ

Theorem 3.1 (Ribet [4] Corollary 1.2).　If every semistable elliptic curve is modular then Fermat’s Last Theorem is true.
定理3.1 (リベット [4] 系1.2)。すべての半安定楕円曲線がモジュラーであればフェルマーの最終定理が真である。

The corollary in [4] states the assumption as all elliptic curves are modular but it suffices to just have semistable elliptic curves be modular.
[4]の系では、すべての楕円曲線はモジュラーであるという仮定を述べていますが、半安定楕円曲線がモジュラーであれば十分です。

Theorem 3.2 (Wiles [6]).　Every semistable elliptic curve over \(\mathbb{Q}\) is modular.
定理3.2 (Wiles [6]). \(\mathbb{Q}\) 上のすべての半安定楕円曲線はモジュラーである。

Note that Wiles’ theorem is independent of the Frey curve, whereas the statement given for Ribet is another general theorem applied to the Frey curve. This summary is about as clean as you can get without expanding further on key concepts like modular forms and Galois representations. In particular, the major contributions of Ribet and Wiles are just short statements for now.
ワイルズの定理はフライ曲線とは独立であるのに対し、リベットの定理はフライ曲線に適用される別の一般定理であることに注意してください。この要約は、モジュラー形式やガロア表現といった重要な概念をこれ以上詳しく説明することなく、可能な限り簡潔にまとめたものです。特に、リベットとワイルズの主要な貢献については、今のところ簡潔な記述にとどめています。

Modular is mentioned but not elaborated in any detail for the time being, one word of warning is that the first definition of modular form one finds is actually not the definition of modular but usually just of level 1. So this initial summary is a proof outline up to a definition of the word modular.
モジュラーについては触れられていますが、今のところ詳しく説明されていません。注意すべき点は、最初に見つかるモジュラー形式の定義は、実際にはモジュラーの定義ではなく、通常はレベル 1 の定義にすぎないということです。したがって、この最初の要約は、モジュラーという単語の定義までの証明の概要です。

3.2 Role of Modular Forms モジュラーフォームの役割

Here we expand on what’s behind the word modular.
ここでは、「モジュラー」という言葉の意味について詳しく説明します。

Key Terms: Conductor, Weight, Level.
キーワード: 導手(導因数)、ウェイト(重さ)、レベル。

3.2.1 Conductor and Point Counts of Semistable Elliptic Curves 半安定楕円曲線の導手と点の数

To keep track of the curve’s reduction behavior across all primes, we have:
すべての素数にわたる曲線の減少挙動を追跡するには、次の式を使用します。

Definition 3.4.　The conductor \(N\) is defined for semistable elliptic curves as the product of all primes where the elliptic curve has bad reduction.
定義 3.4.　半安定楕円曲線の導体 \(N\) は、楕円曲線が悪い約分を持つすべての素数の積として定義されます。

A more general definition exists when the curve is not semistable (which can be found in the appendix), from which, N isn’t necessarily square-free and it being square-free (i.e. \(N\) being the product of distinct primes) is equivalent to the curve being semistable.
曲線が半安定でない場合は、より一般的な定義が存在し (付録に記載されています)、それによれば、N は必ずしも平方自由ではなく、平方自由であること (つまり、\(N\) が異なる素数の積である) は曲線が半安定であることと同等です。

Let \(rad(n)\) denote the product of distinct prime factors of \(n\). Note \(rad(n)\) is square-free for all n, this notation allows the nice expression:
\(rad(n)\) を \(n\) の異なる素因数の積とします。\(rad(n)\) はすべての n に対して平方根を持たないことに注意してください。この表記法によって、次のような簡潔な表現が可能になります。

Proposition 3.3. The conductor of the Frey curve is given by
命題3.3. フライ曲線の導体は次のように与えられる。 \[ N = rad( abc ). \]

Proof. This is immediate from the definition of the conductor above since Frey’s curve is semistable.
証明。フライ曲線は半安定であるため、これは上記の導体の定義から直ちに明らかである。

So, the conductor of the Frey curve is then the product of all primes dividing \(abc\) . Note that \(N\) is even as one of \(a, b, c\) is even.
したがって、フライ曲線の導手は、\(abc\) を割り切るすべての素数の積となります。\(N\)は偶数であり、\(a, b, c\)のいずれかが偶数であることに注意してください。

Now if we let \(E_p\) be the curve defined by the reduction of the Elliptic Curve modulo \(p ∤ ∆\), i.e. where the curve has good reduction, define
ここで、\(E_p\) を楕円曲線の \(p ∤ ∆\) を法として定義される曲線、すなわち曲線が良い縮小を持つ曲線とすると、次のように定義されます。 \[ A_p = p + 1 − |E_p| \] where \(|E_p|\) denotes the number of solutions \((x,y)\) which satisfy the equation of the Elliptic Curve modulo \(p\). If the curve has bad reduction at \(p\) there are 2 cases:
ここで、\(|E_p|\)は、楕円曲線の方程式を法\(p\)で満たす解\((x,y)\)の数を表します。曲線が\(p\)で不完全な縮約を持つ場合、以下の2つのケースが考えられます。

  • • Additive reduction: \(A_p = 0\),
    加法削減: \(A_p = 0\)、

  • • Multiplicative reduction: Ap = ±1, sign known in each case.
    乗法減算：\(A_p = ±1\), それぞれの場合で符号が判明。

We further extend this to the prime powers. Define \(A_1 = 1\), and for the prime powers \(A_{p^k}\) can be defined by the recurrence relation for \(k ≥ 2\)
これを素数冪に拡張する。\(A_1 = 1\) と定義すると、素数冪に対して \(A_{p^k}\) は \(k ≥ 2\) の漸化式によって定義できる。 \[ A_{p^k}= \begin{cases} A_pA_{p^{k−1}} & \text{if \(p\) is of bad reduction} \\ \\ A_pA_{p^{k−1}} − pA_{p^{k−2}} & \text{if \(p\) is of good reduction} \end{cases} \]

We can extend this definition to a general n multiplicatively, e.g. for coprime \(p, q,\) we define \(A_{pq} = A_pA_q\).
この定義を乗法的に一般のnに拡張することができます。たとえば、互いに素な\(p, q,\)に対して\(A_{pq} = A_pA_q\)と定義します。

3.2.2 Preliminary Definition of Modular モジュラーの予備定義

The word modular refers to the ability to associate a modular form.
「モジュラー」という言葉は、モジュラー形式を関連付ける機能を指します。

This amounts to taking the sequence \(A_n\) and defining the following complex valued function
これは、シーケンス \(A_n\) を取り、次の複素数値関数を定義することに相当する。 \[ f(z) = \sum_{n=0}^\infty A_ne^{2πinz} \]

for all \(z ∈ \mathbb{C}\) with strictly positive imaginary part. Let us write \(\mathcal{H}\) for the set of all such \(z\). This definition necessitates the series to converge in \(\mathcal{H}\).
\(z ∈ \mathbb{C}\) の虚部が正であるすべての \(z\) に対して成り立つ。そのような \(z\) 全体の集合を \(\mathcal{H}\) と書く。この定義は、級数が \(\mathcal{H}\) で収束することを要求する。

• It is called a cusp form if \(A_0 = 0\).
\(A_0 = 0\) の場合には cusp 形式と呼ばれます。

We also see that such a function is necessarily periodic, that is \(∀z\) replacing \(z\) with \(z + 1\) changes every exponent by a multiple of \(2πi\) and hence:
また、このような関数は必然的に周期的であることもわかります。つまり、\(∀z\) を \(z\) に置き換えると、すべての指数が \(2πi\) の倍数に変化し、したがって次のようになります。 \[ f(z + 1) = f(z) \] This is a special case of the more general requirement that there exist integers \(k, N\) such that
これは、より一般的な要件である整数\(k, N\)が存在し、 \[ f\left(\frac{az+b}{cz+d}\right)=(cz+d)^k f(z) \] for all \(z ∈ \mathcal{H}\) where \(a, b, c, d ∈ \mathbb{Z}, c = 0 \mod N\), and \(ad − bc = 1\).
すべての\(z ∈ \mathcal{H}\)に対して、\(a, b, c, d ∈ \mathbb{Z}, c = 0 \mod N\)、\(ad − bc = 1\)が成り立ちます。

We see for example, if we take \(a = 1 , b= 1 , c= 0 , d= 1,\) we recover the condition of being periodic.
たとえば、\(a = 1、b= 1、c= 0、d= 1、\) とすると、周期的であるという条件が回復されることがわかります。

The \(N\) is called the level and \(k\) the weight . Other parts of the definition will be introduced in the modular forms section.
\(N\) はレベル、\(k\) は重みと呼ばれます。定義の他の部分はモジュラー形式のセクションで説明します。

3.2.3 Ribet’s Proof Revisited リベットの証明の再考

You might notice that the letter \(N\) is already taken, but this is explained by the following elaboration of Ribet’s contribution:
文字 \(N\) がすでに使われていることに気付いたかもしれませんが、これは Ribet の貢献を詳しく説明した次の説明で説明されています。

  • • It was known that if an elliptic curve is modular, then the level of its modular form, is of the same value, \(N\), as the conductor of the elliptic curve.
    楕円曲線がモジュラーである場合、そのモジュラー形式のレベルは、楕円曲線の導体と同じ値 \(N\) になることが知られています。

  • • Ribet showed that for odd primes \(p \neq P\), if \(p | N\) and \(p^2 ∤ N\) then we can replace our modular form of level \(N\) by another modular form of level \(N/p\) and weight 2.
    リベットは、奇数の素数 \(p \neq P\) に対して、\(p | N\) かつ \(p^2 ∤ N\) であれば、レベル \(N\) のモジュラー形式をレベル \(N/p\) と重み 2 の別のモジュラー形式で置き換えることができることを示しました。

  • • Since for the Frey curve, \(N\) is square free and even, this process can be iterated until \(N = 2\) or \(N = 2P\) with weight 2. If \(P | N\), Ribet removes it using a result of Mazur. So we are left with a level two weight two cusp form.
    フライ曲線の場合、\(N\)は平方自由かつ偶数であるため、この処理は重み2の\(N = 2\)または\(N = 2P\)になるまで反復できます。\(P | N\)の場合、RibetはMazurの結果を用いてそれを削除します。したがって、レベル2、重み2のカスプ形式が残ります。

  • • It was previously known that there are no non-zero cusp forms of level and weight 2.
    レベルと重み 2 の非ゼロのカスプ形式は存在しないことが以前はわかっていました。

We see that revealing the role of modular forms expands the elliptic curve story a little bit more and unlocks a chunk of disproving the Frey curve is modular, while the part about elliptic curves being modular is still basically locked. You might also be wondering how the \(P\) is used. To really unlock these parts of the story, we will need to talk about Galois representations.
モジュラー形式の役割を明らかにすることで、楕円曲線の話が少し広がり、フライ曲線がモジュラーであることを反証する部分が明らかになります。一方、楕円曲線がモジュラーであるという部分は基本的に未解決です。\(P\) がどのように使われるのか疑問に思われるかもしれません。この部分を真に解明するには、ガロア表現について話す必要があります。

3.3 Role of Galois Representations ガロア表現の役割

Keywords (we don’t assume you already know): Galois Representation, \(l\)-adic, Point at Infinity, Abelian Group, Torsion Points
キーワード（既にご存知とは想定していません）: ガロア表現、\(l\)進、無限遠点、アーベル群、捩れ点

We now start mentioning Galois representations somewhat like how modular forms were mentioned in the previous or first subsection, with proper defini- tions deferred to another section.
ここで、モジュラー形式が前の節、つまり最初の節で言及されたのと少し同じようにガロア表現について言及し始めます。ただし、適切な定義は別の節に延期されます。

The word Galois refers to Galois theory, which in turn is named after the mathe- matician Galois who famously died in a duel. We will use Galois representations as a black box in this subsection and try to introduce its origins, the two main types we will consider, as well as their place in the proof.
ガロアという言葉はガロア理論を指し、ガロア理論は決闘で亡くなったことで有名な数学者ガロアにちなんで名付けられました。この節ではガロア表現をブラックボックスとして扱い、その起源、ここで検討する2つの主要な表現、そしてそれらが証明においてどのような位置を占めるかを紹介します。

3.3.1 Every Elliptic Curve has a Galois Representation すべての楕円曲線にはガロア表現がある

We try to introduce the origins of Galois representations from elliptic curves.
楕円曲線からガロア表現の起源を紹介したいと思います。

Given the points of an elliptic curve, there is a meaningful way in which we can add any two points \(A\) and \(B\) among them:
楕円曲線上の点が与えられている場合、それらの点の間に任意の 2 つの点 \(A\) と \(B\) を追加する意味のある方法があります。

  • 1. Take the line \(AB\) and intersect it with the elliptic curve at a third point \(C\).
    直線 \(AB\) を楕円曲線と3番目の点 \(C\) で交差させます。

  • 2. Then take the vertical line through \(C\) and intersect it again with the elliptic curve at \(C^\prime\).
    次に、\(C\)を通る垂直線を引き、\(C^\prime\)で楕円曲線と再び交差させます。

This final intersection point is defined to be \(A + B = C^\prime\). This can be seen in the diagram below with the elliptic curve in green, line \(AB\) in black, and \(A+B\) as the second intersection of the vertical line through \(C\) with the elliptic curve.
この最後の交点は、\(A + B = C^\prime\) と定義されます。これは下の図で、緑色の楕円曲線、黒色の直線 \(AB\)、そして \(C\) を通る垂直線と楕円曲線の2番目の交点である \(A+B\) で示されています。

You may notice that when both points \(A\) and \(B\) line on a vertical line, there is no third point of intersection with the elliptic curve, disrupting the definition of their sum.
両方の点 \(A\) と \(B\) が垂直線上にある場合、楕円曲線との 3 番目の交点は存在せず、それらの和の定義が崩れることに気付くかもしれません。

One way to account for this is to add an extra point \(\mathcal{O}\) called a point at infinity and define the sum of any two vertical points to equal this point.
これを説明する 1 つの方法は、無限遠点と呼ばれる追加の点 \(\mathcal{O}\) を追加し、任意の 2 つの垂直点の合計がこの点と等しくなるように定義することです。

In this way we see that the point \(\mathcal{O}\) behaves like zero in that for all points on the curve
このように、点\(\mathcal{O}\)は、曲線上のすべての点に対して、ゼロのように振舞うことがわかります。 \[ Q + \mathcal{O} = Q. \]

We also see that two points \(Q, R\) in the sense: on a vertical line are negatives of each other
また、2つの点 \(Q, R\) は、垂直線上にあるという意味で、互いに負であることも分かります。 \[ Q + R = \mathcal{O} \]

We are now able to give a working definition for elliptic curve:
これで、楕円曲線の実用的な定義を与えることができるようになりました。

Definition 3.1. An elliptic curve \(E\) over \(\mathbb{Q}\) are all the rational pairs \((x,y)\) satisfying some equation \(y^2 = x^3 + a_2x^2 + a_4x + a_6\) with rational constants \(a_2, a_4, a_6\), together with an extra point \(\mathcal{O}\), called the point at infinity, such that \(∆ \neq 0\).
定義 3.1. \(\mathbb{Q}\) 上の楕円曲線 \(E\) とは、有理定数 \(a_2, a_4, a_6\) を伴う何らかの方程式 \(y^2 = x^3 + a_2x^2 + a_4x + a_6\) を満たすすべての有理数対 \((x,y)\) と、無限遠点と呼ばれる追加の点 \(\mathcal{O}\) で構成され、\(∆ \neq 0\) が成り立ちます。

Any time an addition is defined, we get a notion of scaling by an integer as repeated addition or repeated subtraction:
加算が定義されるたびに、繰り返し加算または繰り返し減算として整数によるスケーリングの概念が得られます。 \[ mQ = \underbrace{Q + Q + ···+ Q}_{m回} \]

We are now ready to define the source of Galois representations:
これでガロア表現のソースを定義する準備が整いました。

Definition 3.2 (m-Torsion Points).
定義 3.2 (m-ねじれ点) \[ E[m] = \{Q ∈ E : mQ = \mathcal{O}\} \]

This torsion is the source of our Galois representations in a similar sense to the sequence of point counts being the raw data for our modular form. This is also where the \(P\) from our initial assumption comes in as we specifically look at the set of \(P\) torsion points, denoted by \(E[P]\).
この捩れは、点数の列がモジュラー形式の生データであるのと同様に、ガロア表現の源泉です。また、\(P\) 捩れ点の集合 \(E[P]\) に注目すると、最初の仮定から \(P\) が登場します。

3.3.2 The Two Types of Galois Representations ガロア表現の2つの種類

Looking at all the powers of a prime \(l\) at once leads to \(l\)-adic Galois representa- tions, while just looking at \(l\) torsion leads to mod \(l\) Galois representation.
素数 \(l\) のすべてのべき乗を一度に見ると \(l\) 進ガロア表現が得られますが、\(l\) 捩れだけを見ると \(\mod l\) ガロア表現が得られます。

3.3.3 Ribet and Wiles revisited リベットとワイルズの再訪

Serre’s full conjecture was that Galois representations under certain conditions were modular and further specified how to get a minimal level and weight.
セールの完全な予想は、特定の条件下でのガロア表現はモジュラーであり、さらに最小レベルと重みを取得する方法を指定するというものでした。

The part about the minimal level and weight is called the epsilon conjecture.
最小レベルと重みに関する部分はイプシロン予想と呼ばれます。

  • • Ribet proves the epsilon conjecture and applies it to the \(\mod P\) Galois representation of Frey’s curve.
    リベットはイプシロン予想を証明し、それをフライ曲線の \(\mod P\) ガロア表現に適用します。

  • • Wiles proves that the 3-adic Galois representation is modular for semi stable elliptic curves and uses known results to conclude that the elliptic curve is modular in general. Part of this was already having the mod 3 Galois representation modular within reach from known results.
    ワイルズは、3進ガロア表現が半安定楕円曲線に対してモジュラーであることを証明し、既知の結果を用いて楕円曲線が一般にモジュラーであると結論付けました。この結論の一部は、既知の結果から、mod 3ガロア表現が既にモジュラーであることが分かっていたことによるものです。

3.3.4 Collection of Key Statements 主要声明集

Here are some examples of the theorems in the actual papers whose statements we hope to understand. We put some here upfront to set our goals for which definitions to include.
実際の論文で述べられている定理の例をいくつか挙げます。これらの定理の記述を理解したいと考えています。どの定義を含めるかという目標を設定するために、いくつか例を挙げておきます。

Example 3.1 (Ribet’s theorem). Ribet typo in actual paper, no one read to end, though at the start there is a typo \(ρ\) that’s supposed to be a \(p\)
例 3.1 (リベットの定理)。実際の論文ではリベットの誤植があり、誰も最後まで読んでいない。冒頭に \(ρ\) と誤植があるが、これは \(p\) であるべきである。

Theorem 8.2. (Main Theorem) Let \(\rho:Gal(\overline{\mathbf{Q}}/\mathbf{Q})→ \mathbf{GL}(2,\mathbf{F})\) be an irreducible mod \(l\) modular representation of level \(Mp\), where \(p\) is a prime not dividing \(M\). Assume that \(\rho\) is finite at \(p\). Then \(\rho\) is modular of level \(M\) provided that at least one of the following two conditions holds:
定理 8.2 (主定理) \(\rho:Gal(\overline{\mathbf{Q}}/\mathbf{Q})→ \mathbf{GL}(2,\mathbf{F})\) をレベル \(Mp\) の既約な mod \(l\) モジュラー表現とします。ここで \(p\) は \(M\) を割り切れない素数です。\(\rho\) は \(p\) で有限であると仮定します。このとき、次の 2 つの条件の少なくとも 1 つが満たされている場合、\(\rho\) はレベル \(M\) のモジュラーです。

(i) The prime \(l\) is a not a divisor of \(M\);
素数 \(l\) は \(M\) の約数ではありません。

(ⅱ) We do not have \(p \equiv 1 (\mod l)\).
\(p \equiv 1 (\mod l)\)は成り立ちません。

We will be able to define everything in this statement by the end except for finite, which is defined in Serre [3] and also showed to be satisfied when needed. Example 3.2 (Theorems in Wiles [6]) . Highlighted in the introduction of [6]
最後に、有限性を除くこの文のすべてを定義できるようになります。有限性はSerre [3]で定義されており、必要に応じて満たされることも示されています。 例3.2 (Wiles [6]の定理) [6]の導入部で強調されています。

(I) \(\rho_0\) is ordinary (at \(p\)) by which we mean that there is a one-dimensional subspace of \(\overline{\mathbf{F}}_p^2\), stable under a decomposition group at \(p\) and such that the action on the quotient space is unramified and distinct from the action on the subspace.
\(\rho_0\) は (\(p\) において) 通常の空間であり、これは \(\overline{\mathbf{F}}_p^2\) の 1 次元部分空間が存在し、\(p\) における分解群の下で安定であり、商空間への作用は分岐せず、部分空間への作用とは異なることを意味します。

(II) \(\rho_0\) is flat (at \(p\)), meaning that as a representation of a decomposition group at \(p,\rho_0\) is equivalent to one that arises from a finite flat group scheme over \(\mathbf{Z}_p\), and det \(\rho_0\) restricted to an inertia group at \(p\) is the cyclotomic character.
\(\rho_0\) は平坦 (\(p\) において) である、つまり、\(p,\rho_0\) における分解群の表現は、\(\mathbf{Z}_p\) 上の有限平坦群スキームから生じる表現と等価であり、\(p\) における慣性群に制限された det \(\rho_0\) は円分特性である。

THEOREM 0.2. Suppose that \(\rho_0\) is irreducible and satisfies either (Ⅰ) or (Ⅱ) above. Suppose also that \(\rho_0\) is modular and that
定理0.2. \(\rho_0\) が既約で、上記の(Ⅰ)または(Ⅱ)のいずれかを満たすとする。また、\(\rho_0\) がモジュラーであり、

(i) \(\rho_0\) is absolutely irreducible when restricted to \(Q\left(\sqrt{(-1)^{\frac{p-1}{2}}p}\right)\)
\(\rho_0\) は \(Q\left(\sqrt{(-1)^{\frac{p-1}{2}}p}\right)\) に制限すると絶対的に既約である。

(ⅱ) If \(q\equiv -1 \mod p\) is ramified in \(\rho_0\) then either \(\rho_0|D_q\) is reducible over the algebraic closure where \(D_q\) is a decomposition group at \(q\) or \(\rho_0|I_q\) is absolutely irreducible where \(I_q\) is an inertia group at \(q\).
\(q\equiv -1 \mod p\) が \(\rho_0\) で分岐している場合、\(D_q\) が \(q\) における分解群であるときに \(\rho_0|D_q\) は代数閉包上で既約であるか、\(I_q\) が \(q\) における慣性群であるときに \(\rho_0|I_q\) は絶対既約であるかのいずれかです。

Then any representation \(\rho\) as in the conjecture does indeed come from a mod- ular form.
すると、予想にあるような任意の表現 \(\rho\) は確かにモジュラー形式から来ていることになります。

THEOREM 0.3. Suppose that \(E\) is an elliptic curve defined over \(\mathbf{Q}\) and that \(\rho_0\) is the Galois action on the 3-division points. Suppose that \(E\) has the following properties:
定理0.3. \(E\) が \(\mathbf{Q}\) 上に定義された楕円曲線であり、\(\rho_0\) が3分割点上のガロア作用であるとする。\(E\) が以下の性質を持つとする。

(i) \(E\) has good or multiplicative reduction at 3.
\(E\) は 3 で良好な、または乗法的縮小を持ちます。

(ⅱ) \(\rho_0\) is absolutely irreducible when restricted to \(\mathbf{Q}\left(\sqrt{-3}\right)\).
\(\rho_0\) は、\(\mathbf{Q}\left(\sqrt{-3}\right)\) に制限される場合、絶対に既約ではありません。

(ⅲ) For any \(q\equiv -1 \mod 3\) either \(\rho_0|D_q\) is reducible over the algebraic closure or \(\rho_0|I_q\) is absolutely irreducible.
任意の \(q\equiv -1 \mod 3\) に対して、\(\rho_0|D_q\) は代数閉包上で既約であるか、\(\rho_0|I_q\) は絶対的に既約ではありません。

Then \(E\) should be modular.
そうすると\(E\)はモジュラーになるはずです。
